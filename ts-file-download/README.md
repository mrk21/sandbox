# CSV generation and downloading

## Dependencies

- Node.js: 20.x
- Next.js: 14.x
- Papa Parse: 5.x
  - CSV Parcer
- encoding.js: 2.x
  - Encoding Converter
- asdf
  - Multiple Runtime Version Manager

## Getting Started

```bash
asdf install
npm install
npm run dev
open http://localhost:3000
```
